<h1>About this project</h1>

<p>This is a basic course managing site, where students enroll into a course, teachers that have access to creating a course, and admins who have the ability to manage users from the admin panel.</p>

<h3>This project includes the following features</h3>
<ul>
    <li>Login/Registering</li>
    <li>List of courses</li>
    <li>User permisions</li>
    <li>Categories</li>
    <li>Admin panel</li>
</ul>

<hr>

<h3>How it was made</h3>

<p> I created the website with HTML CSS PHP bootstrap, you can look at the git history for the site development progression.</p>
